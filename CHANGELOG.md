# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-23
### Added
- Fabric client mod base for Minecraft 1.21.11.
- Clickable nickname moderation flow from chat.
- Action selection UI for mute, warn, ban, and ipban.
- Duration input screen using `d` and `h` format.
- Reason input/preset flow.
- Built-in moderation presets based on RaidMine rules 2.1-3.10.
- Command generation for `/mute`, `/warn`, `/ban`, `/ipban`.
- GitHub-ready repository structure.
